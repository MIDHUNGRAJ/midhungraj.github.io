# Linear Regression Explained

Linear regression is a statistical method used to model and understand the relationship between variables:
- One or more independent variables (X)
- One dependent variable (Y)

Linear regression finds a line that best fits the dataset. This line helps predict the dependent variable (Y) based on the independent variable(s) (X).

## Use Cases

1. Predicting House Prices
2. Stock Price Prediction
3. Medical Cost Estimation
4. Sales Forecasting

## Key Points

- Y: the values you want to predict
- X: the values used to make the prediction

## Mathematical Interpretation

The equation for a simple linear regression is:

```
Y = wX + b
```

Where:
- w (weight) is the slope
- X is the independent variable
- b (bias) is the y-intercept

## Basic Example

```python
import numpy as np

X = np.array([1, 2])
Y = np.array([300, 500])

# The X variable contains only one feature

w = 200  # slope
b = 100  # y-intercept

predicted_Y = w * X + b
```

In this example, we've chosen values for w and b because we know that when we use the equation Y = wX + b, we get the correct Y values:

- (200 * 1) + 100 = 300
- (200 * 2) + 100 = 500

This results in a perfect prediction for Y. However, it's important to note that this is a very basic example. Normally, we find w and b using a cost function, as they are hyperparameters that need to be optimized.

## Visualization

To visualize the regression line:

```python
import matplotlib.pyplot as plt

plt.scatter(X, Y, marker="X", c="g", label="Actual Value")
plt.plot(X, predicted_Y, label="Our prediction")
plt.legend()
plt.show()
```

## Conclusion

Now you understand how regression works to find values for Y. When we provide a value for X, the corresponding Y value will be determined based on the line of best fit.

Remember that in real-world scenarios, we typically use optimization techniques and cost functions to find the optimal values for w and b, rather than setting them manually as in this simple example.
